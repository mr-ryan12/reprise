# Reprise

A modern Phish concert discovery app built with React Router v7, TypeScript, Prisma, PostgreSQL, and Railway.

## Planned Stack
- React Router v7
- TypeScript
- Prisma
- PostgreSQL
- Axios
- Tailwind
- shadcn/ui

## Development Tooling
- Claude Code
- GitHub SpecKit

## Planned Features
- Search Phish shows
- View show details
- Save favorites
- Future AI/MCP-powered discovery
